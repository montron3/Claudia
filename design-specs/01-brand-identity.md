### Logo Design Principles

## üìù Typography

### Font Families

**Primary Font: Inter** (or similar modern sans-serif)
- **Rationale**: Excellent readability, professional yet friendly, optimized for screens
- **Alternatives**: System UI fonts, SF Pro, Roboto
- **Weights**: 400 (regular), 500 (medium), 600 (semibold), 700 (bold)
- **Usage**: All UI text, body copy, headings

**Monospace Font: JetBrains Mono** (or similar)
- **Usage**: Code, fiscal codes, IBAN numbers, technical data
- **Weights**: 400 (regular), 500 (medium)

### Typography Guidelines

**Readability Best Practices:**
- Maximum line length: 70-80 characters for body text
- Use adequate line height (1.5-1.75 for body copy)
- Maintain clear visual hierarchy through size and weight
- Use sentence case for UI labels (not ALL CAPS)
- Left-align text in Italian (natural reading direction)

**Emphasis Techniques:**
- **Bold (600 weight)**: Important terms, fiscal amounts
- *Italic*: Minimal use, legal disclaimers
- Color: Use semantic colors for status/category
- Size: Use type scale, avoid arbitrary sizes

- Use "tu" form (not "lei" formal)

- Clear disclaimers when needed
- Explain the "why" behind requirements

## üìê Design Principles

### 1. Clarity Over Cleverness
Focus on making information easy to understand rather than visually impressive.

### 2. Progressive Disclosure
Show essential information by default, reveal complexity on demand.

### 3. Consistent Patterns
Use the same UI patterns for similar actions across the platform.

### 4. Accessible by Default
Design for all users, including those with disabilities.

### 5. Mobile-First Thinking
Design for mobile viewports first, enhance for larger screens.

### 6. Performance & Speed
Optimize for fast load times and responsive interactions.
